One-time Google setup required

  1. Go to console.cloud.google.com → create or select a project
  2. Enable the Google Docs API and Google Drive API
  3. Go to Credentials → Create → OAuth 2.0 Client ID
  4. Choose Web application, add http://localhost:3001/auth/callback as an Authorized redirect URI
  5. Download the JSON and save it as server/credentials.json
  6. Then click the button below to log in with your Google account
